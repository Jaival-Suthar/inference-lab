# Roadmap

`inference-lab` is starting with a stable local CLI and a persistent benchmark history.
The next steps stay intentionally high level so the public release remains easy to understand.

## Near Term

- add support for additional local runtimes
- deepen command-line reporting
- make comparison workflows more discoverable

## Mid Term

- add a dashboard for exploring stored benchmark runs
- improve export and analysis workflows
- expand the documentation with more examples and troubleshooting notes

## Long Term

- keep the workspace architecture flexible enough to adopt new runtimes and storage backends
- preserve a single mental model across the CLI, metrics, and reporting layers

The roadmap is intentionally concise. It is a guide for direction, not a promise of exact sequencing.
