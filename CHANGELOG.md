# Changelog

All notable changes to `inference-lab` will be recorded in this file.

The project is releasing publicly at `v1.0.0`.

## [1.0.0] - 2026-07-23

### Added

- Ollama-backed inference client
- terminal CLI for local completions and benchmarks
- streaming inference support
- benchmark summaries and metric formatting
- SQLite persistence for benchmark runs
- documentation, release metadata, and repository hygiene files

### Notes

- The first public release keeps the architecture intentionally small.
- Future runtime adapters, dashboard work, and broader reporting remain outside the scope of this tag.
