# Documentation

This directory contains the durable, user-facing documentation for `inference-lab`.
Start here when you want a higher-level explanation than the root README provides.

## Index

- [Architecture overview](architecture.md) - how the CLI, client, metrics, and storage packages fit together.
- [CLI reference](cli-reference.md) - every public command with examples.
- [Benchmark methodology](benchmark-methodology.md) - how benchmark numbers are collected and interpreted.
- [Runtime configuration](runtime-configuration.md) - environment variables and defaults.
- [Troubleshooting](troubleshooting.md) - common startup and connectivity issues.

## Writing Guidelines

- Keep documentation aligned with the shipped CLI.
- Prefer short examples that can be copied directly into a terminal.
- Avoid implementation details unless they help a reader understand the public behavior.
